GET and POST

GET

  • Parameters are passed in the URL
  • The browser can cache your request parameters in the history
  • GET can only send text data
  • In PHP to access the GET variable we use $_GET
  • You can send a GET request from the browser using the URL with parameters

POST

  • Parameters are passed in the request body
  • The browser does not cache your request parameters in the history
  • POST can send binary data
  • In PHP to access the POST variable we use $_POST
  • You can send a POST request from an application such as POSTMAN and attach the parameters in the body

Postman: It is a tool used for testing and debugging APIs
