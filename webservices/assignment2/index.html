Session: A session in PHP is a variable assigned to a client, which can be accessed across multiple web pages (APIs). This can be used for authenticating the user, each time a user interacts with the API.

Each client has a unique session on the server

We have the following APIs

  1. Authenticate (Login)
  2. Logout
  3. Check Authentication Status

Steps to establish a Session in PHP:

  • Step 1) User sends the username and password to the server
  • Step 2) The server checks if the username and password matches the record from the database
  • Step 3) If the record does not match the users table in the database, we give an error response.
    If it matches, we proceed to step 4
  • Step 4) The server issues a cookie to the browser of the client containing the session ID. In correspondence to the session ID, the server stores the variable which can be checked for authentication
  • Step 5) All requests from the client would have the cookie, which would be used by the server to authenticate the client
